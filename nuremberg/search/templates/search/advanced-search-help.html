Most advanced queries can be made naturally. For example, to find English-language documents about Erhard Milch relating to treatment of workers in 1944, search for:

workers Erhard Milch english 1944 Search

By default, documents that contain all keywords in any field are found. Our data can sometimes be incomplete, so you should usually try a more general search first. The filtering options shown on the left will let you narrow down the results.

However, you can also define specific fields to use for keyword search. For example, to restrict your search to documents written by Milch, search for:

workers english 1944 author:Erhard Milch Search

Or to include documents used in Milch's prosecution, but written by someone else, you can exclude a field using the minus sign:

workers english 1944 defendant:milch -author:milch Search

You can search for any number of exact phrase matches in or out of a field by quoting the phrase to be searched, such as:

"international law" Search

Field search can include several keywords. If you want to search across all fields after adding a fielded search, you can use the special "all" field, so these are equivalent:

"international law" trial:NMT 4 Search

trial:NMT 4 all:"international law" Search

For many fields, we have incomplete data, and you can specify only documents without a field using the special keyword "unknown" or "none":

workers date:unknown Search

instructions trial:none Search

You can search within any field listed in the filters to the left. Valid field names are: type, trial, author, defendant, date, language, source
